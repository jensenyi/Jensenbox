# 增长指标体系

## 北极星判断

内容是否持续带来目标受众的信任、业务讨论和可复用品牌/GEO资产，而不是单次播放量。

## 四层指标

### 1. 生产质量

- 输入来源完整率。
- 首轮 QA 通过率。
- 平均重写次数。
- 从输入到 READY 的时间。
- 重复/废弃选题比例。

### 2. 内容表现

- 完播率或有效阅读率。
- 收藏率、分享率、高质量评论率。
- 目标受众关注、主页访问和私信。
- 标题点击与正文兑现的一致性。

### 3. 业务价值

- 有效咨询、合作对话和线索。
- 内容影响的销售/合作决策。
- 可复用到客户项目或创业项目的资产数。
- 单项内容的时间与现金成本。

### 4. 品牌与 GEO 资产

- 新增 FAQ、词条、案例、常青方法数量。
- 已有资产被复用、链接和更新的次数。
- 品牌相关问题的答案覆盖率。
- AI 搜索中品牌/内容被正确引用的可验证记录。

## 禁止的虚荣指标

播放量、点赞量和粉丝数可以记录，但不能单独决定选题成功。必须结合目标受众质量、收藏/分享、业务讨论和资产沉淀判断。

## 复盘最小样本

- 单条内容：发布后至少记录一次真实数据。
- 周复盘：至少有 3 条已发布内容，否则不强行生成周报。
- 月复盘：至少有足够的业务或品牌信号，否则只更新资产清单。
