# 系统清单

版本：0.4.0 MVP

## 已完成

- 品牌、受众、内容规则和增长指标。
- 七个 Agent 的生产级 Prompt。
- 统一状态机、数据契约、交接和重写机制。
- 每日 Run 初始化、状态更新和校验工具。
- 评分卡、发布表、复盘模板和 GEO 资产目录。
- 三个低空转模块线程。
- 一个由原影视创作线程迁移而来的项目总控中台。
- 影视创作知识库挂载及视频内容生产质量门。
- 从 200+ Agency Agents 中筛选 15 个岗位方法，并压缩植入现有总控与七个 Agent。
- 核心上下文包、按需读取和引用式交接的 Token 节流协议。
- C 盘 Codex 项目到 D 盘实际工作区的目录联接。

## 当前边界

- 不自动抓热点。
- 不自动登录、发布或操作平台账户。
- 不创建周期性 Token 自动化。
- 第一次真实运行需要填写 `runs/2026-06-21/00_input.md`。

## 下一版本触发条件

只有完成至少 3 次真实 MVP Run 后，才评估：

- 是否接入自动来源采集。
- 是否增加 Airtable/数据库看板。
- 是否增加半自动发布包。
- 是否调整质量权重和平台策略。

没有真实使用证据，不扩建系统。
