# 大川品牌档案

## 品牌主体

- 对外名称：大川 / Jensen Yi
- 当前角色：品牌增长操盘手、创业者、AI 时代增长实践者
- 长期方向：AI 品牌负责人、能够把 AI 变成品牌与增长基础设施的人
- 非目标角色：AI 工具测评博主、知识付费导师、流量娱乐博主

## 一句话定位

大川研究并实践：品牌如何在 AI 搜索、智能内容和新消费环境中获得可持续增长。

## 核心价值主张

1. 不追逐工具热度，关注工具如何改变用户决策和商业结果。
2. 不只讲传播，关注内容如何沉淀为品牌、GEO 和增长资产。
3. 不输出空洞方法论，用案例、机制和可执行动作说话。
4. 以初创团队视角追求高性价比、可复制、可验证。

## 品牌人格

- 有审美，但不装腔。
- 有判断，但不故作权威。
- 懂增长，也尊重长期品牌价值。
- 对 AI 乐观，但反对为了 AI 而 AI。
- 像亲自干过项目的负责人，而不是旁观评论员。

## 内容支柱

| 优先级 | 内容支柱 | 核心问题 |
|---|---|---|
| P0 | 品牌增长 | 品牌动作如何影响认知、转化和复购？ |
| P0 | GEO / AI 搜索 | 品牌如何被 AI 理解、引用和推荐？ |
| P0 | 国际消费品牌案例 | 大品牌的动作如何拆成小团队可复制机制？ |
| P1 | 创业与商业模式 | 小团队如何用有限资源建立增长飞轮？ |
| P1 | 用户增长与私域 | 如何让流量变成关系和复利资产？ |
| P1 | AI 生产力 | AI 如何真正减少成本、提高质量或改变流程？ |

## 可信度来源

- 真实项目经验和品牌操盘视角。
- 对中国内容平台语境、口语、情绪和用户表达的理解。
- ChatGPT、Grok、图像模型、Seedance 等 AI 工具的工作流实践。
- GEO、内容生产、视觉传播和多平台品牌曝光经验。

## 禁止表达

- “颠覆一切”“遥遥领先”“普通人最后机会”等无证据大词。
- 把单个平台数据包装成普遍规律。
- 没有项目依据却使用“我操盘过”“我们验证过”。
- 用工具清单替代商业判断。
- 为制造冲突而虚构立场或贬低从业者。

## 视觉基调

- 潮流、编辑感、强视觉层级、黑/米白/蓝为核心色。
- 可使用粗粝纸张、胶带、手写标记、信息卡和大号编号。
- 不做普通 PPT 风、廉价科技蓝、满屏小字和模板化渐变。
