# 受众档案

## 核心受众 A：消费品牌增长负责人

- 年龄参考：28-42 岁。
- 场景：要做增长、内容和新品，但团队小、预算紧、内部协作慢。
- 真问题：热点很多但无法转成品牌动作；内容产量高但资产少；AI 工具多但流程没有变。
- 需要：可复制案例、决策框架、执行模板、风险提醒。
- 反感：新闻搬运、宏大叙事、只讲曝光、不讲转化和沉淀。

## 核心受众 B：创业者和中小企业老板

- 场景：本人就是产品、品牌和销售的最终负责人。
- 真问题：没有完整团队；每个动作都要考虑现金流和投入产出。
- 需要：低成本试验、最小可行方案、优先级和停止条件。
- 反感：大企业专属方案、需要十几个岗位配合的学院派框架。

## 核心受众 C：AI 时代的品牌/内容从业者

- 场景：担心被工具替代，也想建立新的职业能力。
- 真问题：会用工具但不会把工具接入业务目标。
- 需要：从需求到交付的工作流、质量标准、案例和方法资产。
- 反感：Prompt 炫技、模型参数测评、没有客户价值的自动化。

## 决策触发点

- “这个案例我能不能在本周复用？”
- “这件事会不会影响品牌被 AI 推荐？”
- “投入多少，多久能知道有效？”
- “哪些动作是品牌资产，哪些只是短期流量？”
- “我该先做哪一步？”

## 内容消费偏好

- 先给结论，再讲机制和案例。
- 一条内容聚焦一个矛盾或一个决定。
- 喜欢结构图、前后对比、拆解清单和真实页面。
- 接受专业概念，但必须翻译成人话。
- 最后需要明确的“今天可以做什么”。

## 平台差异

| 平台 | 主要人群状态 | 更适合的表达 |
|---|---|---|
| 抖音 | 快速判断是否值得继续看 | 3 秒冲突、单一观点、可视化证据 |
| 小红书 | 搜索、收藏、对比方案 | 清晰标题、步骤、模板、审美图卡 |
| 视频号 | 熟人信任、老板与管理者语境 | 稳健判断、案例复盘、经营视角 |
| 公众号/文章 | 深度理解和长期搜索 | 完整论证、来源、方法与案例 |
