# Agency Agents 精选知识源

本目录保存从 200+ 专业岗位中筛出的 15 份原始角色定义。它们是方法参考，不是 15 个新线程，也不会独立运行。

## 选择标准

1. 直接服务内容工厂 MVP 的输入、判断、生产、质量或资产沉淀。
2. 能形成明确交付物或质量标准。
3. 与现有能力互补，而不是制造重复岗位。
4. 适合初创团队低成本复用。

## 使用规则

- 实际执行以 `docs/agency_agents_integration.md` 的压缩职责卡为准。
- 原岗位中的平台规则和行业数字可能过时，使用前需核验。
- 原岗位里的宏大 KPI 不直接作为大川项目目标；至少完成 3 次真实 Run 后再建立基线。
- 不因知识源增加而新增线程、会议、日报或自动化。
- 日常 Run 不读取 `source/` 原始岗位文件，只使用 `docs/agency_agents_integration.md`；原文仅用于规则升级或具体争议。
